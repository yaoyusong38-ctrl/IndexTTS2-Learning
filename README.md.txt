# IndexTTS2-Learning 🎤

本项目记录了我在本地部署 **IndexTTS2 文本转语音（Text-to-Speech, TTS）模型** 的学习过程与实践结果。

---

## 🚀 启动方式
 1️⃣ 安装依赖
```bash
pip install -r requirements.txt


2️⃣ 运行项目
uv run webui.py --fp16


运行后看到：

Running on local URL: http://0.0.0.0:7860


打开浏览器访问：
👉 http://localhost:7860

即可进入 IndexTTS2 的 WebUI 界面。
